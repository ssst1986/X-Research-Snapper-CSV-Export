# X Research Snapper & CSV Export 📌

[日本語は英語の後にあります / Japanese follows English]

A professional productivity tool for marketers and researchers to turn X (formerly Twitter) into a structured database.

## 🚀 Key Features (English)
- **Noise-Free Mode**: Automatically hides "For You" tabs, trends, and ads.
- **One-Click Snap**: Click the "📌 収穫(Harvest)" button to stock tweets in the side panel.
- **Data Export**: Export stocked data to CSV with metrics like engagement rates.
- **Hunter Mode**: Automatically ranks tweets as "Legendary," "Viral," or "Target" based on impressions.

---

## 🚀 主な機能 (Japanese)
- **Noise-Free Mode**: 「おすすめ」タブ、トレンド、広告を自動で非表示。
- **One-Click Snap**: 投稿にある「📌 収穫」ボタンでサイドパネルに保存。
- **Data Export**: 反応率などの指標を含めたCSV出力。
- **Hunter Mode**: インプレッションに基づき、投稿を「超弩級」「大バズ」等に自動ランク付け。

## 🛡️ Privacy & Security / プライバシー
- **Local Only**: All data is stored in your browser's Local Storage.
- **No External Sync**: No data is ever sent to external servers.
- **Open Source**: Full source code is available here for transparency.

## 🛠 How to Install / インストール方法
1. Download this repository as a ZIP and unzip it.
2. Open `chrome://extensions/` in Chrome.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the unzipped folder.
