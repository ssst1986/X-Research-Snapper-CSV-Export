# X Research Snapper & CSV Export 📌

X（旧Twitter）でのリサーチを資産に変える、マーケター・リサーチャー向け生産性向上ツール。
ノイズを排除し、重要な投稿を一瞬でストック、分析可能なCSVデータとして出力します。

## 🚀 主な機能

- **Noise-Free Mode**: 「おすすめ」タブ、トレンド、広告を自動で非表示。必要な情報だけに集中できます。
- **One-Click Snap**: 投稿にある「📌 収穫」ボタンを押すだけで、サイドパネルに投稿をストック。
- **Data Export**: ストックした投稿データをワンクリックでCSV出力。出力後はリストのリセットも可能です。
- **Focus Player**: リサーチを邪魔しない小型YouTubeプレイヤーを内蔵。URL入力で動画の切り替えが可能です。
- **Hunter Mode**: インプレッション数に基づき、投稿を「超弩級」「大バズ」「観測対象」に自動ランク付け。

## 🛡️ プライバシーとセキュリティ

- **完全ローカル動作**: 収集したデータはすべて、ブラウザ内のローカルストレージにのみ保存されます。
- **外部送信なし**: 外部サーバーへのデータ送信は一切行いません。
- **オープンソース**: 全てのソースコードを本リポジトリで公開しています。

## 🛠 インストール方法 (開発者モード)

現在、Chrome ウェブストアの審査中です。手動でテストする場合は以下の手順で行ってください。

1. 本リポジトリの右上にある「Code」ボタンから **Download ZIP** を選択し、解凍します。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパー モード」をオンにします。
4. 「展開して読み込む」をクリックし、解凍したフォルダを選択します。

## 📊 CSV出力項目の詳細

出力されるCSVには以下の項目が含まれます（最新版 v1.2 対応）：
- **保存日時 / 投稿日時**
- **ユーザー名 / 投稿本文 / URL**
- **表示回数 (Impressions)**
- **反応率 (%)**: (いいね + リポスト) ÷ 表示回数 で算出
- **いいね数 / リポスト数**
- **ランク**: 拡散規模に応じた自動判定
